# 什么是扩展 API？

嘉立创 EDA 专业版扩展 API（下称「扩展 API」）是一套建立在前端的 JS API 接口，用于开发嘉立创 EDA 专业版编辑器的各种自定义功能。简而言之，扩展 API 采用 JavaScript 的基础语法编写源代码，对众多嘉立创 EDA 专业版的功能和算法进行了高度封装，配合完善的 [SDK](./ancillary-projects/pro-api-sdk)、[类型约束](./ancillary-projects/pro-api-types)，只需要很少的精力就可以完成自定义扩展功能的开发，有效降低开发者和设计者的心智负担。

接下来，你可以前往 [如何开始](./how-to-start) 章节，了解如何从零开始安装环境并构建一个自己的嘉立创 EDA 专业版扩展。

如果你是扩展用户，请参阅 [扩展的获取和使用](../user-guide/using-extension) 章节。
